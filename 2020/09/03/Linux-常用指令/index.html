tar命令

参数

  • c 压缩文件
  • x 解压文件
  • z 格式为gzip
  • v 显示执行文件列表
  • f 要操作的文件

直接解压

tar –zxvf xxx.tar / xxx.tar.gz

解压后，编译安装
